happy (đăng ký + xác nhận) · error (password policy, email trùng, link hết hạn) · edge (resend cooldown 60s/5-ngày)
1signup → 2verify-email-sent → 3verify-email-success → 4verify-email-expired

1login [chung]

Flow 5 — Quên mật khẩu / đặt lại

happy (đặt lại, thu hồi session) · error (link hết hạn 30ph, password policy) · edge (anti-enumeration)
1forgot-password-request → 2reset-password-form

happy (gỡ khi đã có password) · edge (chưa có password → buộc tạo trước)
